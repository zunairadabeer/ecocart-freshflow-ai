# EcoCart FreshFlow AI

Active Project - Currently In Development

EcoCart FreshFlow is a food waste reduction platform designed to help users manage household food inventory, track expiry dates and discover recipes using ingredients they already have.

The project aims to encourage more sustainable food consumption by helping users make better use of the food they purchase before it expires.

## Current Features

* Inventory management
* Expiry date tracking
* Detection of soon-to-expire items
* Recipe recommendations based on available ingredients
* FastAPI backend

## Tech Stack

* Python
* FastAPI
* Scikit-Learn
* SQLite (planned)
* AI-assisted recommendation logic

## Example Workflow

1. User adds food items to their inventory
2. The system tracks expiry dates
3. Items nearing expiry are highlighted
4. Recipe recommendations are generated using available ingredients

## Background

This project was originally created as part of my IEUK Technology & Engineering Internship programme.

As part of the internship, I was challenged to identify a real-world problem and develop a technology-based solution. I chose to focus on household food waste because many people purchase food that ends up being thrown away due to forgotten ingredients, poor inventory tracking or missed expiry dates.

The original concept centred around creating a digital inventory system that could help users keep track of food items and receive suggestions on how to use ingredients before they expire. Since completing the internship, I have continued developing the idea independently and exploring how AI could be integrated to provide smarter recommendations and improve the user experience.
